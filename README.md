# Blind Games

Blind Games is a professional Android application designed specifically for blind users, visually impaired users, and seniors.

## Features

- Full TalkBack support
- Arabic and English
- High contrast mode
- Large fonts
- Voice guidance
- Vibration feedback
- AI opponents
- Offline gameplay

## Games

### Domino

- Egyptian Domino
- All Fives (American Domino)
- Block Domino
- Draw Domino
- Mexican Train

### Other Games

- Chess
- Backgammon
- Card Games

## Accessibility

- Android Accessibility API
- TalkBack
- Braille Display Support
- Keyboard Navigation
- Text To Speech

## Technology

- Kotlin
- Jetpack Compose
- Material 3
- MVVM
- Room Database
